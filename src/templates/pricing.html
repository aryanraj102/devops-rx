Frequently asked

Is Pro really one-time?

Yes. Pay ${{ price }} once and your account has Pro access permanently, including features we add later.

What log formats work?

Any plain-text log. The classifier handles Linux syslog, Apache/Nginx access logs, Kubernetes event logs, Docker logs, and most structured formats.

Do I need to set up Jira and Slack?

Only if you want those integrations. Analysis and checklist generation work without them. Add credentials in your .env file when you're ready.

Which LLM does DevOps-Rx use?

DeepSeek V3 via OpenRouter. Fast, accurate on technical logs, and cost-efficient for large files.
